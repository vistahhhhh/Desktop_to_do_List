# 桌面待办 2.0 — 便签功能 PRD

> 版本: v1.0 | 日期: 2026-03-26 | 状态: 设计中

---

## 一、功能概述

在"桌面待办"应用中集成**轻量便签**功能，用户可以随时记录碎片化想法、参考资料、会议笔记等非结构化内容。便签与待办任务**双向关联**，形成"想法 → 行动"的完整闭环。

### 核心理念

```
便签 = 随时可写的草稿纸
待办 = 必须完成的行动项
链接 = 想法驱动行动，行动回溯想法
```

---

## 二、用户场景

| 场景 | 描述 |
|------|------|
| **快速记录** | 开会时随手记下要点，不想打断当前工作流 |
| **任务备注** | 某个任务需要详细的参考信息（链接、步骤、注意事项） |
| **灵感暂存** | 突然想到一个点子，先记下来，之后再转为待办 |
| **日记/复盘** | 每天简短记录今日感悟，回顾时查看 |



## 布局设置

在左上角“桌面待办”的右侧添加一个“便签”选项

单击可以切换到便签

未选中状态为灰色，字体比选中的“桌面待办”小两号，与“桌面待办”向下对齐而非居中对齐

用户进入时默认为“桌面待办”页面

# “便签”选项卡的设置

便签选项卡的“设置”与桌面待办选项卡共用，最上面一排都共用

下面直接是便签，不用有新的框，直接全是便签内容

便签：支持设置标题和正文，支持基本的加粗、斜体、下划线——这几个像word一样放在页面上方排成一条，

页面最下方原来“新建”的位置是便签列表，点开后出现上拉列表（列表支持滚轮），里面是之前创建的便签。

页面最下方左侧是加号，点击可以添加新便签

页面最下方右侧是回收站


便签输入后不用自己点保存，而是会自动保存到便签列表。如果没有设置标题，取正文前7个字+“…”显示在列表中，支持在列表中对便签进行删除
