# 桌面待办 — 轻量级桌面悬浮任务 & 便签管理工具

> 一款精致小巧的 Windows 桌面悬浮应用，集任务管理与便签速记于一体，帮你随时掌控每一天的工作节奏。

---

## 核心亮点

- **桌面悬浮窗** — 始终置顶、半透明、不遮挡工作区，随时可见待办事项
- **无边框圆角设计** — 现代极简 UI，自由拖拽位置、边缘缩放大小
- **任务 + 便签双模块** — 左侧管理任务，右侧速记便签，两者可一键关联
- **五套主题配色** — 深色 / 浅色 / 纯黑 / 莫兰迪粉 / 莫兰迪绿，一键切换
- **三种任务类型** — 今日待办、本周计划（未来10天）、长期任务，覆盖所有时间尺度
- **系统托盘常驻** — 关闭即隐藏，双击托盘立刻恢复，不占任务栏空间
- **单实例运行** — 重复点击程序自动唤醒已运行的窗口到前台

## 功能详览

### 📋 三大智能清单

| 清单 | 说明 |
|------|------|
| **☀ 今日待办** | 管理当天的短期任务，过期未完成任务自动显示"移交今日"按钮 |
| **▦ 本周计划** | 未来10天的中期任务，可选择具体日期，过期任务提供"重新编辑日期"入口 |
| **◈ 长期任务** | 无固定日期限制的长期目标，支持设定截止日期或仅选月份 |

### ✏️ 新建 & 编辑任务

- 点击底部 **「＋ 添加」** 按钮或右键空白区域快速新建
- 自动匹配当前清单类型（在"今日待办"下新建即为当日任务，在"本周计划"下新建即为周计划）
- 编辑弹窗包含：**标题**、**任务类型**、**截止日期**、**优先级**（高/中/低）、**标签关联**、**详细描述**
- 双击任务卡片即可编辑，支持 Enter 快捷保存

### ✅ 任务状态流转

- 点击任务左侧圆圈一键切换 **待办 ↔ 完成**
- 已完成任务标题自动添加删除线并排到列表底部
- 右键任务卡片可编辑或删除

### 📌 分任务（子任务）

- 每个主任务可展开添加多条分任务，支持折叠/展开
- 分任务独立管理状态（待办 / 完成），可单独删除
- 移交今日时，分任务随主任务一同转移，不会丢失

### 📝 便签模块

- 顶部标签页一键切换 **待办 / 便签** 两大模块
- 便签编辑器支持**富文本**：加粗、斜体、下划线、删除线、有序/无序列表
- 左侧便签列表可快速浏览和切换所有便签
- 支持**拖拽便签到桌面**，生成独立的浮动便签小窗口
- 浮动便签窗口支持置顶、拖拽移动、边缘缩放、关闭后自动保存
- 无内容的便签关闭后自动清理，不会残留空记录

### 🔗 任务 ↔ 便签关联

- 在任务卡片右键菜单中，选择**「新建关联便签」**或**「关联已有便签」**
- 关联后，任务卡片显示链接徽标，便签标题自动添加 `[任务名]` 前缀
- 新建关联便签会自动弹出浮动便签窗口，方便立即记录
- 在便签侧也可通过**「管理关联任务」**对话框搜索并关联任务
- 支持**替换关联**：若目标任务已有便签，确认后自动解除旧关联、清除旧前缀
- 替换时若旧便签无实际内容（仅有标题前缀、正文为空），自动删除该空便签
- 通过任务新建便签但未写任何内容时，关闭后便签不会被保存，任务保持未关联状态
- **移交今日**时，关联便签随任务一同转移到新日期，关联关系完整保留

### 🏷️ 自定义标签系统

- 在左侧标签栏**右键**即可新建标签（鼠标悬停有提示"右键新建标签"）
- 每个标签可选择专属图标（24种精选符号：★ ♦ ● ◆ ▲ ■ 等）
- 标签支持编辑名称、更换图标、删除
- 任务可关联多个标签，点击标签即可筛选查看相关任务

### 🗑️ 回收站

- 删除的任务和便签不会立即消失，进入回收站
- 支持**逐条恢复**或**逐条永久删除**
- 一键**清空回收站**

### 📅 待办回顾

- 查看最近10天的历史每日任务完成情况
- 按日期分组，最新日期排在最上方
- 已完成任务显示 ✓ 标记和删除线，未完成显示 ○
- 支持滚轮浏览，一键**清空历史**

### 🎨 主题与个性化

| 设置项 | 范围 |
|--------|------|
| **主题切换** | 深色半透明 / 浅色半透明 / 纯黑 / 莫兰迪粉 / 莫兰迪绿 |
| **背景透明度** | 30% ~ 100%，滑块实时预览 |
| **字号大小** | 10px ~ 24px，滑块实时预览 |
| **窗口置顶** | 开关控制是否始终置于最前 |

所有设置修改**实时生效**，关闭设置面板自动保存。

### 🖱️ 交互体验

- **拖拽移动** — 按住标题栏拖拽窗口到任意位置
- **边缘缩放** — 鼠标移到窗口边缘/角落，拖拽调整大小
- **拖拽排序** — 长按任务卡片拖拽调整顺序
- **拖拽便签到桌面** — 从便签列表拖出便签，生成独立浮动窗口
- **多选操作** — 任务列表和便签列表均支持 Ctrl+单击 切换选中、Shift+单击 范围选中
- **批量删除** — 多选后按 Delete 键或右键菜单批量删除
- **点击空白取消选中** — 点击列表空白区域即可取消所有选中状态
- **位置记忆** — 窗口位置、大小在下次启动时自动恢复
- **系统托盘** — 最小化/关闭按钮将窗口隐藏到托盘，双击托盘图标恢复

## 安装方式

### 方式一：安装程序（推荐）

运行 **桌面待办_安装程序.exe**，按向导安装：
- 自动创建桌面快捷方式和开始菜单项

直接运行 `dist\桌面待办\桌面待办.exe`，无需安装，开箱即用。

## 使用技巧

1. **快速新建**：右键任务列表空白处，选择"新建任务"
2. **快速完成**：点击任务左侧的状态圆圈，一键标记完成
3. **移交遗留**：昨天没做完的任务会自动出现在今天，点击"移交今日"即可（分任务和关联便签会一同转移）
4. **关联便签**：右键任务 →「新建关联便签」，边做边记
5. **拖出便签**：在便签列表中拖拽便签到窗口外，生成桌面浮动便签
6. **批量管理**：Ctrl/Shift 多选任务或便签，Delete 键一键批量删除
7. **标签筛选**：点击左侧自定义标签，只看该标签下的任务
8. **托盘操作**：右键托盘图标可快捷新建任务、打开设置
9. **不会重复打开**：再次双击程序，已运行的窗口会自动跳到前台
